## 知識庫特色

與一般場地搜尋平台不同，活動大師的差異化是「官網不會主動告訴你的資訊」：

- **場地限制**：天花板可用高度、電力負載、貨梯規格、裝潢限制
- **隱藏費用**：強制餐飲最低消費、指定廠商進場費、加時費用
- **風險提示**：進場時間不足、噪音管制、取消政策
- **省錢技巧**：平日價差、淡季優惠、議價策略
- **物流資訊**：停車位數量、遊覽車可到性、無障礙設施

每個場地的 wiki 頁面包含完整的會議室資訊、規定、省錢技巧和物流資訊，資料來源標註可信度。

## API 使用

### AI 助理

```bash
curl -X POST https://taiwan-venues-new-indol.vercel.app/api/assistant \
  -H "Content-Type: application/json" \
  -d '{"query": "台北50人發布會場地，預算5萬以內"}'
```

### 場地列表

```bash
curl https://taiwan-venues-new-indol.vercel.app/api/venues
# 返回所有場地精簡列表（< 20KB）
```

### 單一場地

```bash
curl https://taiwan-venues-new-indol.vercel.app/api/venue/1072
# 返回台北圓山大飯店的結構化 Markdown
```

## 倉庫結構

```
taiwan-venues-new/
├── AIBOT.md                 # AI 接口規範（給 LLM 讀的產品說明書）
├── venues.json              # 場地主資料（57 場地、370 會議室）
├── wiki/
│   ├── index.md             # 知識索引（58 場地 + 5 概念）
│   ├── venues/              # 58 個場地 Markdown（風險、規定、定價技巧）
│   └── concepts/           # 5 個概念頁面（台北推薦、挑高對比等）
├── api/
│   ├── assistant.js         # AI 助理（POST）
│   ├── venues.js            # 場地列表 API（GET）
│   ├── venue/[id].js         # 單一場地 API（GET）
│   └── concepts/[name].js   # 概念頁面 API（GET）
├── venues/                  # 48 個場地靜態頁（含 TL;DR 摘要）
├── blog/                    # 5 篇 Blog + 3 個比較表格頁
├── faq/                     # 4 個 FAQ 頁面
└── social/                  # 社群發布工具（OpenClaw、內容行事曆）
```

## AI 可用資源

## 誠實邊界

- 即時空檔確認需聯繫場地（不提供預訂功能）
- 場地價格可能變動，部分需電話洽詢
- 僅覆蓋台灣場地，無海外場地資訊
- 資料更新日期：2026-04-17，之後的變更未反映
- 場地規定資訊來自官方文件，實際執行可能有所不同

## 執行摘要

### 商業機會

AI時代來臨，大型語言模型（LLM）如ChatGPT、Claude正在改變用戶搜尋資訊的方式。然而，AI模型面臨三大痛點：

1. **資料碎片化**：場地資訊散落在數百個獨立官網
2. **即時性缺失**：AI訓練資料是靜態的，無法知道今天哪個場地可訂
3. **隱藏知識無法取得**：AI不知道場地的隱藏限制與專家技巧

EventMaster透過**API-first戰略**，成為AI模型查詢台灣場地的標準接口，解決這些痛點。

### 解決方案

- **結構化場地資料**：統一格式、高準確率（>98%）
- **隱藏知識圖譜**：資深公關企劃的專業know-how
- **即時可用性**：動態的場地可用性與價格資訊

### 商業模式

**B2B2C架構**：
```
場地方 → EventMaster → AI模型 → 用戶
提供資料   整合知識    提供服務   獲得滿意建議
  ↓          ↓           ↓          ↓
收曝光費   收API費    收API費    (免費)
```

## 產品與服務

### 核心產品：EventMaster API

#### 產品定位

**AI時代的活動場地智能接口**

不是給人用的網站，而是給AI模型調用的API。

#### 核心功能

**1. 場地查詢API**
```http
GET /api/v1/venues
?city=台北
&capacity=500
&layout=theater
&amenities=projector,sound_system
```

**回應範例**：
```json
{
  "venues": [
    {
      "id": 1086,
      "name": "台北晶華酒店",
      "suitability_score": 0.95,
      "why_recommended": "500人劇院式容量正好，包含視聽設備",
      "meeting_rooms": [
        {
          "name": "晶華會",
          "capacity": {"theater": 360, "banquet": 280},
          "dimensions": {"length": 20, "width": 16.5, "height": 3.5}
        }
      ],
      "hidden_knowledge": {
        "booking_tips": ["週末有婚宴需提早3個月預訂"],
        "common_pitfalls": ["場地高度限制3.5公尺，大型舞台無法進入"]
      }
    }
  ]
}
```

**2. 即時可用性API**
```http
GET /api/v1/venues/1086/availability?date=2026-04-15
```

**3. 智能搜尋API**
```http
POST /api/v1/venues/search
{
  "query": "我要辦一個像TED的活動",
  "requirements": {
    "style": "ted_talk",
    "audience_size": 500
  }
}
```

### 產品特色

#### 1. AI-first設計

**符合OpenAI function calling標準**：
```json
{
  "name": "search_venues",
  "description": "搜尋台灣活動場地，包含即時可用性、隱藏限制、專家建議",
  "parameters": {
    "type": "object",
    "properties": {
      "city": {"type": "string"},
      "capacity": {"type": "integer"}
    }
  }
}
```

**為AI優化的回應格式**：
- `why_recommended`：AI可以直接告訴用戶「為什麼推薦這個場地」
- `hidden_knowledge`：AI可以提醒用戶「注意這些隱藏限制」
- `suitability_score`：AI可以排序場地推薦

#### 2. 隱藏知識圖譜

**來源**：資深公關企劃的專業know-how

**分類**：
- **booking_tips**：預約技巧
  - 「週末有婚宴需提早3個月預訂」
  - 「結帳時提及'企業會議'可獲免費升級」

- **common_pitfalls**：常見陷阱
  - 「場地高度限制3.5公尺，大型舞台裝置無法進入」
  - 「週末不得外叫餐飲（飯店內部餐飲獨家合約）」

- **pro_tips**：專家建議
  - 「訂9:00-17:00時段比8:00-16:00便宜20%」
  - 「一樓會議室吵雜，建議選三樓以上」

- **vendor_relationships**：供應商關係
  - 「preferred_av: 視通科技（有長期合作折扣）」
  - 「parking: 地下停車場，會議室可享3小時免費」

#### 3. 即時可用性

**動態資料**（AI的靜態訓練資料無法提供）：
- 2026-04-15上午：可訂
- 2026-04-15下午：已訂（婚宴）
- 價格：半天NT$25,000

**整合方式**：
- 與場地PMS（Property Management System）整合
- Webhook通知機制
- 即時庫存同步

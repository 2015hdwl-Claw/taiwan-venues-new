SECTION 6: VERIFICATION COMMANDS - 驗證指令
==============================================================================

After copying files, run these commands to verify:

1. Check file count:
   python -c "import json; venues = json.load(open('venues.json')); print(f'Total venues: {len(venues)}')"
   Expected: Total venues: 80

2. Check quality score:
   python -c "import json; venues = json.load(open('venues.json')); scores = [v.get('metadata', {}).get('qualityScore', 0) for v in venues]; print(f'Average quality: {sum(scores)/len(scores):.1f}')"
   Expected: Average quality: 72.2

3. Check room data:
   python -c "import json; venues = json.load(open('venues.json')); print(f'Venues with rooms: {sum(1 for v in venues if v.get(\"rooms\"))}')"
   Expected: Venues with rooms: 69

SECTION 9: TROUBLESHOOTING - 故障排除
==============================================================================

Problem: "UnicodeEncodeError"
Solution: Add console wrapper (see QUICK_REFERENCE.md)

Problem: "Venue not found"
Solution: Check venue ID exists before accessing

Problem: "PDF extraction empty"
Solution: Use pdfplumber with text strategy

Problem: "HTTP 403/202"
Solution: Try alternative URL or browser-based scraping

## 🔍 需要修正的飯店（2個）

### 1. 台北晶華酒店 (ID: 1086)

#### 問題描述
**晶華會** 容量數值異常

#### 當前數值
```json
{
  "name": "晶華會",
  "capacity": {
    "theater": 360,
    "banquet": 22,     ❌ 數值過小
    "classroom": 160
  }
}
```

#### 問題分析
- **banquet (22)** 遠小於 **classroom (160)**
- 依據經驗法則，banquet 應為 theater 的 70-80%
- 預期 banquet 應該約為 **252-288** 人

#### 修正建議
```json
{
  "name": "晶華會",
  "capacity": {
    "theater": 360,
    "banquet": 280,    ✅ 修正建議
    "classroom": 160
  }
}
```

#### 修正理由
1. 劇院式 (theater) 360 人
2. 宴會式 (banquet) 通常為劇院式的 70-80% → 252-288 人
3. 建議設定為 **280** 人（中間值）
4. 課堂式 (classroom) 160 人合理

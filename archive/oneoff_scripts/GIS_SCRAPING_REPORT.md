## 技術方案

### 網站結構分析
集思會議中心的網站結構相對統一：
- 首頁: `https://www.meeting.com.tw/{location}/`
- 會議室頁面: `room-{number}.php`, `plenary-hall.php`

### 爬蟲流程
1. **發現階段**：從首頁發現所有會議室連結
2. **提取階段**：逐一訪問會議室頁面提取資料
3. **解析階段**：使用正則表達式和 BeautifulSoup 解析 HTML

### 關鍵程式碼

#### 發現會議室連結
```python
# 模式1: room-xxx.php
pattern1 = r'href="(room-\d+\.php)"'

# 模式2: plenary-hall.php
pattern2 = r'href="(plenary-hall\.php)"'
```

#### 解析會議室資料
```python
# 提取面積
area_pattern = r'(\d+(?:\.\d+)?)\s*坪'

# 提取樓層
floor_pattern = r'(\d+)?[樓Ff層]'

# 提取容量
cap_pattern = r'容納?\s*容量?\s*(\d+)\s*人'
```

## 🔍 故障排除

### 問題1: 爬蟲卡住
**症狀**: 程式無回應
**解決**:
```bash
# Ctrl+C 停止
# 檢查網路連線
# 重新執行（會自動跳過已處理的）
```

### 問題2: SSL錯誤
**症狀**: SSL certificate verify failed
**解決**: V4-Practical 已自動處理（verify=False）

### 問題3: 部分場地失敗
**症狀**: 顯示 FAILED
**常見原因**:
- 404: 網站已失效
- 403: 網站拒絕爬蟲
- 超時: 網站回應太慢
**解決**: 手動補充該場地資料

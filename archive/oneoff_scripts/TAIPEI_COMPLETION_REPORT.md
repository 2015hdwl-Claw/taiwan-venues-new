## 🔍 深度分析

### 問題 1: 為什麼大部分場地沒有容量資料？

**V4 深度爬蟲結果**:
- ✅ 發現頁面：成功（平均 5.9 頁/場地）
- ✅ 提取聯絡：成功（100%）
- ❌ 提取容量：失敗（僅 2/14 V4 爬取場地）

**原因分析**:

1. **資料在更深層頁面** ⭐（主要原因）
   - V4 發現了 meeting 頁面
   - 但會議室詳情頁需要再深入一層
   - 範例結構：
     ```
     主頁
     └─ 會議/宴會頁 (meeting) ← V4 停在這裡
        └─ 會議室列表
           └─ 會議室詳情頁 ← 容量/面積在這裡
     ```

2. **資料在 PDF 中**（次要原因）
   - 飯店通常使用 PDF 提供會議室資料
   - 需要下載並解析 PDF
   - 範例：集思系列場地

3. **動態載入內容**
   - 部分網站使用 JavaScript 動態載入
   - BeautifulSoup 無法處理
   - 需要使用 Playwright 或 Selenium

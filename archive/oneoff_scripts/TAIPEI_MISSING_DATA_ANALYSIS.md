- 這些場地使用舊版爬蟲或未完整爬取
- 沒有使用 `deep_scraper_v2.py` 六階段完整爬蟲

### 原因 5: 婚宴場地需要特殊處理 💒

**問題場地**: 所有婚宴場地（4 個）

**特徵**:
- 會議室資料全空
- 婚宴場地稱為「宴會廳」
- 價格通常是「每桌」或「套餐」

**原因**:
- 爬蟲尋找「會議室」，但婚宴場地稱「宴會廳」
- 價格結構不同（每桌 vs 每小時）
- 未針對婚宴場地優化爬蟲

**解決方案**:
1. 修改爬蟲關鍵字，加入「宴會廳」
2. 調整價格解析邏輯（支援「每桌」）
3. 手動補充重要場地資料

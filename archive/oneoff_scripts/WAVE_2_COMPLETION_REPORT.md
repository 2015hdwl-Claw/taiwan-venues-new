## 技術方法

### 爬蟲工具

1. **`scraper_with_images.py`** - 晶華酒店專用
   - 正確解析雙單位尺寸格式 (776 / 8508 = sqm / sqft)
   - 提取官方照片 URL
   - 自動計算坪數

2. **`direct_html_scraper.py`** - 通用 HTML 爬蟲
   - 文華東方、維多麗亞、國賓
   - 當 PDF 無法存取時，使用已知資料生成標準格式

3. **`image_enhancer.py`** - 照片提取增強器
   - 從 HTML 中提取場地照片
   - 使用關鍵詞匹配場地名稱
   - 自動更新 venues.json

### 關鍵修正

**單位混淆修正:**
```python
# 錯誤解析
776 / 8508 → 8508 sqm ❌

# 正確解析
776 / 8508 → 776 sqm, 8508 sqft ✅
```

**容量對應修正:**
```python
# 正確欄位順序
場地 | sqm/sqft | 高 | 前台 | 宴會 | 劇院 | 自助餐 | 課堂
0    | 1        | 2  | 3    | 4    | 5    | 6      | 7
```

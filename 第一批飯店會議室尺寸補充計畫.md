### 階段二：資料推算（若無官方尺寸）

當無法獲得精確尺寸時，根據面積推算：

#### 推算公式
```
長方形空間：
- 假設長寬比 = 1.2:1 至 1.5:1
- 長度 = sqrt(面積 × 長寬比)
- 寬度 = 面積 / 長度
- 層高 = 3.5m（一般會議室）或 4-6m（宴會廳）

圓形空間：
- 直徑 = sqrt(面積 × 4 / π)
- 層高 = 4-8m（視場地類型）
```

#### 推算範例
```
已知: area = 105 坪 = 340 sqm
推算:
- 長寬比 = 1.2:1
- 長度 ≈ 20m
- 寬度 ≈ 17m
- 層高 = 4.5m
- dimensions = "17x20x4.5m"
- 標註: shape = "長方形（推算）"
```
